# 04 — 迭代优化：设计你的专属卡片

> 🕐 预计用时：20 分钟

核心功能已经有了。现在是最自由的环节——**设计你自己的卡片风格**，让每个人的插件都独一无二。

### 🗺️ 优先级指南

| 优先级 | 内容 | 建议 |
|--------|------|------|
| 🌟 **必做** | 选一个卡片模板风格（5 选 1） | 这是本章核心，每个人做出不同的风格 |
| ⭐ 推荐 | 自定义水印/署名 | 简单又有个人标记感 |
| 💫 进阶 | 快捷键 / 深浅色切换 | 提升使用体验 |
| 🎁 探索 | AI 背景 / 多尺寸 / 统计 | 时间充裕再玩 |

> 💡 20 分钟建议集中做好 1 个风格。贪多嚼不烂，做精一个比每个碰一点效果好。

---

## 🎨 卡片模板风格（🌟 必做：5 选 1）

每个人选一个自己喜欢的方向，用对话告诉 Claude Code：

### 小红书风格

```
帮我做一个"小红书风格"的卡片模板：
- 卡片改为 1:1 正方形（适合发朋友圈）
- 奶油色背景
- 选中文字用珊瑚粉色（#FF6B6B）高亮
- 左侧引用竖线也改为珊瑚粉
- 底部加一行 "via @我的名字" 作为个人署名
- 圆角更大一些
```

### 暗黑学术风

```
帮我设计一个"暗黑学术风"卡片：
- 深色背景（#1a1a2e）
- 高亮文字用金色（#FFD700）
- 上下文文字用暗灰色（#555）
- 左侧引用线用金色
- 底部信息用细的金色分隔线
- 整体感觉像精装书的内页
```

### 便利贴风

```
做一个"便利贴"风格：
- 淡黄色背景（#FFF9C4），有轻微的纸张纹理感
- 文字用手写风的衬线字体
- 没有引用竖线，改为上下用引号 "" 包裹选中文字
- 右下角画一个图钉装饰
- 底部信息更小更淡
```

### 极简主义

```
做一个极简风格：
- 纯白背景，大量留白
- 只显示选中的文字（不要上下文）
- 文字居中，24px 大字号
- 底部只有一行小字：来源标题
- 没有二维码，没有装饰，只有文字的力量
```

### 赛博朋克

```
做一个赛博朋克风格的卡片：
- 深黑色背景
- 高亮文字用霓虹绿（#00FF41），要有发光效果（text shadow）
- 上下文用深灰色
- 加一些装饰性的线条和网格
- 二维码也用绿色配色
- 底部信息栏有一个扫描线动画效果（静态图里做不了动画，那就用线条模拟）
```

**自由发挥！** 你完全可以自己想一个风格，用对话描述给 Claude Code。

## 🖼️ AI 生成背景（选做）

如果想让卡片更有视觉冲击力：

```
给卡片添加 AI 生成背景功能：
- 分析选中文字的关键词，用英文拼一个 prompt
- 用 Pollinations.ai 生成一张抽象/模糊的背景图
  URL: https://image.pollinations.ai/prompt/{prompt}?width=720&height=400&nologo=true
- 把背景图画在 Canvas 最底层，上面加半透明遮罩确保文字可读
- 这个功能可选，在右键菜单里加一个子菜单：
  a. "✂️ 生成卡片（纯色背景）"
  b. "🎨 生成卡片（AI 背景）"
```

---

## ⌨️ 快捷键（选做）

```
给划词卡片添加键盘快捷键：
- 选中文字后按 Ctrl+Shift+S（Mac 上是 Cmd+Shift+S）直接生成卡片
- 不用右键，更快
- 在 manifest.json 的 commands 里配置
```

---

## 📐 多种尺寸（选做）

```
支持多种卡片尺寸，在右键菜单里可选：
- 竖版（720x自适应）— 适合微信聊天
- 方形（720x720）— 适合朋友圈/小红书
- 横版（1080x608）— 适合 Twitter/微博
```

---

## 🌙 深色/浅色模式（选做）

```
给卡片加一个深色/浅色模式切换：
- 在 popup 里加一个开关
- 浅色模式：白底深字
- 深色模式：深色底浅色字
- 用户偏好保存到 chrome.storage
```

---

## ✏️ 自定义水印/署名（选做）

```
在 popup 的设置里加一个"个人署名"输入框：
- 用户可以输入自己的名字或社交媒体 ID
- 卡片底部显示 "via @xxx"
- 保存到 storage，下次打开还在
```

---

## 📊 使用统计（选做）

```
在 popup 里显示一些有趣的统计：
- 总共生成了多少张卡片
- 最常在哪些网站使用（Top 3 网站）
- 平均每天生成几张
- 用简单的柱形图或数字展示
```

## 描述效果的技巧

当卡片样式不满意时：

**对比法**：
```
现在的上下文文字太灰了看不清，我希望能比现在深一点但还是要和高亮文字有区分
```

**参考法**：
```
我想让整个卡片的感觉更像微信读书的划线分享
```

**渐进法**：
```
引用线太细了 → 粗一倍 → 再粗一点，用渐变色 → 完美
```

**回退法**：
```
刚才的修改不好看，帮我撤销
```
