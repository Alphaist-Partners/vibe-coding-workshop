✨ 功能特点

🖱️

选中即生成

选中文字 → 右键 → 卡片图片自动复制到剪贴板

🎨

精美排版

高亮金句、淡化上下文、引用竖线、来源水印一应俱全

📱

QR 码溯源

卡片自带页面二维码，扫码即可跳转原文

📋

一键粘贴

生成后自动复制，直接 Ctrl+V 粘贴到微信、飞书、笔记

🗂️

历史记录

所有生成的卡片自动保存，随时回顾和再次复制

🌙

多种风格

极简白、暗黑、小红书、赛博朋克…总有一款适合你

🚀 安装方法

  1. 下载项目

    克隆或下载本仓库到本地

    git clone https://github.com/YOUR_USERNAME/share-card.git
  2. 打开 Chrome 扩展页

    地址栏输入 chrome://extensions，打开右上角「开发者模式」

  3. 加载扩展

    点击「加载已解压的扩展程序」，选择项目文件夹

  4. 开始使用

    在任意网页选中文字 → 右键 →「📌 生成分享卡片」
