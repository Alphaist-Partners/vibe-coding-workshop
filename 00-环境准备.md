# 00 — 环境准备

> 🕐 预计用时：15 分钟

---

## 1. 确认 Chrome 浏览器

你几乎肯定已经有了。没有的话：[google.com/chrome](https://www.google.com/chrome/)

### 开启"开发者模式"

我们写的插件还没发布到 Chrome 商店，需要以"开发者模式"加载：

1. 在 Chrome 地址栏输入：`chrome://extensions/`
2. 右上角打开 **开发者模式** 开关
3. 看到出现 "加载已解压的扩展程序" 按钮就对了

> 💡 这个步骤 macOS 和 Windows 完全一样。

---

## 2. 安装 Node.js

Claude Code 需要 Node.js 运行。

### macOS

```bash
brew install node

# 没有 Homebrew？先装它：
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### Windows

访问 [nodejs.org](https://nodejs.org)，下载 **LTS 版本**，安装时保持默认选项。

### 验证

```bash
node --version   # 应该输出 v18.x 或更高
```

---

## 3. 安装 Claude Code

```bash
npm install -g @anthropic-ai/claude-code
```

### 验证 & 首次配置

```bash
claude --version
claude            # 首次运行会引导登录
```

> ⚠️ Claude Code 需要 Anthropic API key。如果团队有共享的 key，教练会现场提供。

---

## 4. 安装 Git

### macOS

```bash
git --version        # 通常已预装
# 如果没有：
brew install git
```

### Windows

访问 [git-scm.com/downloads](https://git-scm.com/downloads/win)，下载安装，保持默认选项。

### 首次配置

```bash
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"
```

---

## 5. 安装代码编辑器（可选）

推荐 **VS Code**：[code.visualstudio.com](https://code.visualstudio.com/)

安装后终端输入 `code .` 可以打开当前文件夹。

---

## 6. 创建项目文件夹

```bash
# macOS
mkdir -p ~/Projects/share-card
cd ~/Projects/share-card

# Windows (PowerShell)
mkdir -Force "$env:USERPROFILE\Projects\share-card"
cd "$env:USERPROFILE\Projects\share-card"
```

---

## 7. 注册 GitHub 账号

后面我们会把代码推到 GitHub 上分享。如果还没有账号：

1. 访问 [github.com](https://github.com)，点击 **Sign up**
2. 跟着引导完成注册（需要验证邮箱）
3. 记住你的用户名，后面会用到

> 💡 已有账号的直接跳过。
