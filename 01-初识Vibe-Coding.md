# 01 — 初识 Vibe Coding

> 🕐 预计用时：20 分钟

---

## 什么是 Vibe Coding？

**Vibe Coding** 是 Andrej Karpathy（OpenAI 联合创始人）在 2025 年初提出的概念：

> "你完全沉浸在氛围（vibe）中，拥抱指数级增长，忘掉代码的存在。就是看着东西跑起来，如果出了问题就把报错贴给 AI，通常它就修好了。"

简单说：**你说人话，AI 写代码，你看效果，不满意就继续说。**

| | 传统编程 | Vibe Coding |
|---|---------|-------------|
| 核心技能 | 记语法、写逻辑 | 描述需求、判断结果 |
| 工作方式 | 手写代码 | 对话驱动 |
| 门槛 | 需要系统学习 | 会说话就行 |
| 适合场景 | 大型系统 | 原型、工具、探索 |

## 认识 Claude Code

Claude Code 是我们今天的"编程搭子"。它运行在终端里，你用自然语言告诉它想做什么，它来写代码。

### 启动

```bash
cd ~/Projects/share-card   # macOS
cd $env:USERPROFILE\Projects\share-card   # Windows PowerShell

claude
```

## 第一次 Vibe：最简单的 Chrome 插件

在 Claude Code 的对话框里输入：

```
帮我创建一个最简单的 Chrome 浏览器插件：
- 点击插件图标后弹出一个小窗口
- 窗口里显示一个大标题 "✂️ 划词卡片"
- 下面一行小字 "选中网页文字，一键生成分享卡片"
- 再下面放一段使用说明："在任意网页选中文字 → 右键 → 生成卡片"
- 用 Chrome Extension Manifest V3 格式
- 整体风格简洁好看，用深色主题
```

### 会发生什么？

Claude Code 会创建几个文件：

| 文件 | 角色 | 类比 |
|------|------|------|
| `manifest.json` | 插件的配置清单 | 产品说明书 |
| `popup.html` | 弹窗的结构 | 房子的框架 |
| `popup.css` | 弹窗的样式 | 房子的装修 |
| `popup.js` | 弹窗的逻辑 | 房子的电路 |

### 加载你的第一个插件

1. 打开 Chrome，地址栏输入 `chrome://extensions/`
2. 确认 **开发者模式** 已开启
3. 点击 **"加载已解压的扩展程序"**
4. 选择你的项目文件夹 `share-card`
5. 点击浏览器工具栏的 🧩 拼图图标，找到你的插件
6. 📌 点"固定"把它钉在工具栏

点击插件图标——你的第一个浏览器插件诞生了！🎉

> 💡 **刷新插件**：每次修改代码后，去 `chrome://extensions/` 点击插件卡片上的 🔄 按钮，然后重新点击插件图标。

## 📝 本章要点

- Vibe Coding = 用自然语言描述需求，AI 来写代码
- Claude Code 是运行在终端里的 AI 编程助手
- Chrome 插件 = manifest.json + HTML + CSS + JS
- 修改后在 `chrome://extensions/` 刷新即可看效果
