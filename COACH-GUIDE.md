# 🧑‍🏫 教练手册

> 面向工作坊教练的教学建议和常见问题处理。

---

## 课前准备

### 环境预检（至少提前 1 天）

- [ ] 所有学员电脑已安装 Chrome 浏览器
- [ ] 所有学员电脑已安装 Node.js 18+
- [ ] Anthropic API key 已准备好
- [ ] 每个学员有 GitHub 账号（提前让大家注册）
- [ ] 网络稳定（QR code API + 可选的 Pollinations.ai 需要联网）
- [ ] 提前完整跑一遍流程，记录可能的卡点
- [ ] 准备好参考实现（reference/ 目录），用于演示最终效果

### 项目选择理由（破冰时可以讲）

为什么选"划词卡片"：

1. **实用性**：做完真的会用，不是一次性玩具
2. **跨平台零差异**：Chrome 在所有系统上一样
3. **零依赖**：不需要 Python、不需要服务器
4. **视觉成果感**：看到自己生成的卡片，成就感很强
5. **个性化空间**：每个人可以设计不同风格，课堂不无聊
6. **完整闭环**：开发 → Git → 部署 → 分享 全流程覆盖

### 备选方案

如果 QR code API（`api.qrserver.com`）不可用：
- 方案 B：内嵌一个轻量 QR code JS 库（让 Claude Code 去找一个）
- 方案 C：不画二维码，底部只显示 URL 文字

如果 Pollinations.ai 不可用（AI 背景功能）：
- 这是选做功能，跳过即可
- 或用 `https://picsum.photos/720/400?random=N` 占位

## 教学节奏

| 环节 | 时间 | 教练动作 |
|------|------|---------|
| 破冰 & 介绍 | 10 min | 演示最终成品——选中文字，右键，卡片出现 |
| 环境准备 | 15 min | 现场演示，逐步指导 |
| 初识 Vibe Coding | 20 min | 先演示，再让学员动手 |
| Git 入门 | 20 min | 用生活类比讲解 |
| 构建插件 | 40 min | **核心**：每步先演示，学员跟做 |
| ☕ 休息 | 10 min | 巡场答疑，帮落后的学员追进度 |
| 迭代优化 | 20 min | **放手**：让学员自选风格，自由发挥 |
| 发布分享 | 15 min | 帮助解决 GitHub 认证问题 |
| 项目主页 | 25 min | 演示 GitHub Pages 配置流程 |
| 总结 & Q&A | 15 min | 收集反馈 |

**总计约 3 小时**（含休息）

### 教学策略

1. **先演示，再动手**：每步先在投影上做一遍
2. **容许差异**：每个人的 Claude Code 输出不同，完全正常
3. **出错是好事**：错误是教学机会，演示如何用 AI debug
4. **巡场辅导**：构建插件阶段（第 03 章）需要频繁巡场
5. **鼓励个性化**：迭代优化阶段让每个人做不同风格，最后互相展示
6. **刷新提醒**：反复提醒修改后要刷新插件

### 课堂氛围提升

- 迭代优化阶段结束后，让每个人展示自己的卡片风格——**这是课堂高光时刻**
- 可以设一个"最佳设计奖"
- 让大家互相试用对方的插件

## Chrome 插件调试三招

一定要教会学员这三个操作：

1. **刷新插件**：`chrome://extensions/` → 🔄 按钮
2. **看网页 Console**：F12 → Console（content script 错误在这里）
3. **看后台错误**：`chrome://extensions/` → "Service Worker" 或 "错误"

## 常见问题

### 环境问题

| 问题 | 解决 |
|------|------|
| Claude Code 安装失败 | 检查 Node.js 版本 ≥ 18 |
| 插件加载失败 | 检查 manifest.json 格式；确认选对了根文件夹 |
| 看不到插件图标 | 🧩 → 固定插件到工具栏 |
| 弹窗一闪就消失 | 右键弹窗 → 检查，DevTools 会固定弹窗 |

### 插件功能问题

| 问题 | 解决 |
|------|------|
| 右键菜单不出现 | 确认 manifest.json 有 contextMenus 权限；刷新插件 |
| 选中文字后菜单没有"生成卡片" | 确认选中了文字（不是点击了图片之类的） |
| 卡片空白 | Canvas 绘制可能报错，看 Console |
| 剪贴板复制失败 | 需要 clipboardWrite 权限；某些页面可能限制 |
| 二维码不显示 | api.qrserver.com 可能被墙，换备选方案 |
| 卡片上文字乱码/重叠 | Canvas 文字换行逻辑有 bug，让 Claude Code 修 |
| 图片粘贴到微信显示不了 | 某些应用不支持从剪贴板粘贴 PNG，试试其他应用 |

### Git & GitHub Pages 问题

| 问题 | 解决 |
|------|------|
| push 需要认证 | 用 `gh auth login` 或 Personal Access Token |
| GitHub Pages 没生效 | 确认 Settings → Pages 配置了 `/docs` 目录 |
| Pages 显示 404 | 确认 docs/ 里有 index.html；等 2 分钟重试 |
| Pages 没更新 | push 后需要 1-2 分钟部署；清浏览器缓存 |

## 评估关注点

重在体验，不是考核：

- ✅ 成功加载插件并看到弹窗
- ✅ 在网页上成功生成了卡片
- ✅ 卡片成功复制到剪贴板
- ✅ 设计了自己的卡片风格（个性化）
- ✅ 至少 4 次 Git 提交
- ✅ 推送到 GitHub
- ✅ GitHub Pages 上线（加分项）
