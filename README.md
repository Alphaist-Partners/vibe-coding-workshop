# ✂️ Vibe Coding 工作坊：划词卡片 Chrome 插件

> **用对话写代码，从零构建一个真正能用的浏览器工具**

## 这是什么

一个面向零技术背景同事的 Vibe Coding 上手工作坊（约 3 小时）。你将通过与 AI（Claude Code）对话，从零构建一个真正实用的 Chrome 浏览器插件——**划词卡片**。

在任意网页上选中一段文字，右键一点，自动生成一张精美的分享卡片：

- ✨ 选中文字高亮显示，上下文适度暗化
- 📰 自动附上文章标题和来源
- 📱 文章链接生成二维码，扫一扫就能看原文
- 📋 卡片自动复制到剪贴板，直接去微信/飞书粘贴

**做完你真的会每天用它。**

## 为什么是这个项目？

- 🔥 **实用** — 不是玩具，做完真的会用
- 🎨 **可个性化** — 每个人设计自己的卡片风格
- 🌍 **跨平台零差异** — Chrome 在 macOS 和 Windows 上完全一致
- 📦 **零后端** — 不需要装 Python、不需要启动服务器
- 🌐 **含网页部署** — 还会做一个项目介绍主页，部署到互联网上

## 技术栈（不用提前学，AI 会帮你写）

- **HTML / CSS / JS** — 网页三件套
- **Chrome Extension Manifest V3** — 最新的插件标准
- **Canvas API** — 浏览器内置的画图能力
- **Chrome Storage API** — 浏览器本地存储
- **GitHub Pages** — 免费的静态网页托管
- **Git** — 代码版本管理
